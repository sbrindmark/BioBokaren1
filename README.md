# BioBokaren2

BioBokaren2 är ett konsolprogram för att boka biobiljetter, visa tillgängliga filmer, hantera studentrabatt och skriva ut kvitto.

## Funktioner

- Lista tillgängliga filmer och tider
- Välj film, tid och antal biljetter
- Aktivera eller ta bort studentrabatt med rabattkod
- Automatisk prisberäkning inklusive moms och eventuell rabatt
- Skriv ut kvitto med alla detaljer

## Användning

1. Starta programmet.
2. Välj ett alternativ i menyn:
    - **1:** Visa tillgängliga filmer
    - **2:** Välj film, tid och antal biljetter
    - **3:** Aktivera eller ta bort studentrabatt (rabattkod: `STUDENT2025`)
    - **4:** Skriv ut kvitto
    - **5:** Avsluta programmet
